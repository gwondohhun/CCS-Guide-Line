Siemens — V-version
Syngo MR VB / VD / VE software
Operation Steps
1
Exam Registration
Patient → Register → 필수 항목(굵은 글씨) 모두 입력 → Confirm → 원하는 프로토콜 선택 후 << 클릭하여 이동
2
DICOM Export
완료된 시리즈 선택 → Transfer 메뉴에서 DICOM 전송 설정 확인 후 전송
3
MPR
3D 데이터셋 우클릭 → MPR/MIP 선택 → 축 방향 조정
4
MIP
TOF/CE-MRA 시리즈에서 MIP 생성 → 슬래브 두께·방향 설정
5
Composing
다중 슬랩 시리즈를 하나로 합성 (Whole Spine 등에 사용)
6
ADC
DWI 완료 후 ADC map 자동 생성 확인; 없으면 후처리 메뉴에서 수동 생성
7
Subtraction
Pre/Post contrast 시리즈 선택 → Image Math → Subtraction
8
minIP
MRCP 등에서 최소값 투사 영상 생성

Planning — Brain
두개저(foramen magnum) ~ 두정부(vertex)
Purpose
  • 뇌 구조이상, 뇌혈관 질환, 염증 질환
  • 외상 후 손상 평가, 신경계 증상 원인 확인
  • 수술 전/후 평가, 전이, 선천성·퇴행성 질환
Coverage
  • 두개저(foramen magnum) ~ 두정부(vertex)
  • 대뇌·소뇌·뇌간(midbrain, pons, medulla) 포함
Plan 기준선
  • AX — Anterior commissure - Posterior commissure Line (AC-PC line) 기준
  • SAG — 정중시상면(midline sagittal) 정렬
  • COR — AC-PC line에 수직
  • OBL COR — Hippocampus 장축 평행 (해마 평가 시)
  • Brain MRA(TOF) — Circle of Willis 중심; slab 경계 조정으로 venetian blind 방지
  • Veno(MRV) — Superior sagittal sinus ~ Jugular vein 커버

🪄 Operation Manual
👌 Planning
🔺 Siemens — V-version
Operation Steps
1
Exam Registration — Patient → Register → 필수 항목(굵은 글씨) 모두 입력 → Confirm → 원하는 프로토콜 선택 후 << 클릭하여 이동
2
Dicom Export — 완료된 시리즈 선택 → Transfer 메뉴에서 DICOM 전송 설정 확인 후 전송
3
MPR — 3D 데이터셋 우클릭 → MPR/MIP 선택 → 축 방향 조정
4
MIP — TOF/CE-MRA 시리즈에서 MIP 생성 → 슬래브 두께·방향 설정
5
Composing — 다중 슬랩 시리즈를 하나로 합성 (Whole Spine 등에 사용)
6
ADC — DWI 완료 후 ADC map 자동 생성 확인; 없으면 후처리 메뉴에서 수동 생성
7
Subtraction — Pre/Post contrast 시리즈 선택 → Image Math → Subtraction
8
minIP — MRCP 등에서 최소값 투사 영상 생성

🧠 Planning — Brain
Purpose
  • 뇌 구조이상, 뇌혈관 질환, 염증 질환
  • 외상 후 손상 평가, 신경계 증상 원인 확인
  • 수술 전/후 평가, 전이, 선천성·퇴행성 질환
Coverage
  • 두개저(foramen magnum) ~ 두정부(vertex)
  • 대뇌·소뇌·뇌간(midbrain, pons, medulla) 포함
Plan 기준선
  • AX — Anterior commissure - Posterior commissure Line (AC-PC line) 기준
  • SAG — 정중시상면(midline sagittal) 정렬
  • COR — AC-PC line에 수직
  • OBL COR — Hippocampus 장축 평행 (해마 평가 시)
  • Brain MRA(TOF) — Circle of Willis 중심; slab 경계 조정으로 venetian blind 방지
  • Veno(MRV) — Superior sagittal sinus ~ Jugular vein 커버

🦴 Planning — C-Spine
Purpose
  • 신경 구조 평가, 퇴행성 변화
  • 외상·종양·염증·감염, 신경학적 증상 원인 확인
  • 추간판탈출증 등 병변 진단 및 추적관찰
Coverage
  • 두개저(foramen magnum)부터 T1–T2 상부까지
  • 경추 7개(C1–C7) 전부 포함

  • SAG — C3~4 기준 / Vertebral Body와 CSF 사이 중간선
  • AX — 각 Disc에 평행하게 기울임
  • OBL — Foramen 평가 시 사선 방향 적용

🦴 Planning — Whole Spine
Coverage
  • C1 ~ S2 전체 (2~3 station)
  • 각 station 간 충분한 overlap 확보 필수
Plan 기준선
  • SAG — 각 station 별 척추 중심선 정렬; OS 70%이상
  • COR — 척추 전체 정렬 확인; OS 70%이상
  • Station overlap — 인접 station 간 overlap 충분히 확보 (Insufficient overlap artifact 방지)

💪 Planning — Shoulder
Purpose
  • 회전근개 파열, 연부조직 손상, 관절염, 활막염, 종양 등 병변 진단 및 추적관찰
Coverage
  • 견관절(Glenohumeral joint) — 관절와(Glenoid fossa)·상완골두(Humeral head) 중심
  • 견봉(Acromion), 쇄골(Clavicle) 원위부
  • 회전근개 근·건 전체 (Supra/Infra/Subscapularis/Teres minor)
  • 관절와순(Glenoid labrum), 상완이두건 장건(Biceps long head)
  • 견봉하·견봉쇄골관절, 삼각근(Deltoid muscle) 포함
Plan 기준선
  • AX — AC joint ~ Proximal humeral 전체 포함
  • COR — Supraspinatus muscle 또는 Joint에 평행
  • SAG — Supraspinatus muscle 또는 Joint에 수직 (교차)
